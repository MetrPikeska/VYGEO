VYGEO OPALENA - Interaktivní mapová aplikace pro ski areál
============================================================

O PROJEKTU
----------
VYGEO je profesionální webová aplikace vytvořená pro ski areál Opalena v Beskydech. 
Jedná se o pokročilou interaktivní mapovou aplikaci s real-time funkcionalitami, 
která kombinuje geografické informační systémy (GIS), počasí, GPS a umělou inteligenci 
pro monitoring a správu lyžařského areálu.

HLAVNÍ FUNKCE
-------------

1. INTERAKTIVNÍ MAPA
   - Vlastní mapové dlaždice s vysokým rozlišením
   - Více podkladových vrstev (mapy.cz, ortofoto, turistická, WMS)
   - Překryvné vrstvy (webkamery, vleky, potrubí, ortofoto)
   - Drag & drop funkcionalita pro GeoJSON soubory
   - Přizpůsobitelné pořadí vrstev

2. SPRÁVA GEOGRAFICKÝCH OBJEKTŮ
   - Vytváření polygonů, linií a bodů
   - Editace a mazání existujících objektů
   - Automatické získávání nadmořských výšek z OpenElevation API
   - Výpočet ploch, délek a převýšení
   - Správa barev z předdefinované palety
   - Ukládání do MySQL databáze

3. REAL-TIME MONITORING
   - Live počasí z OpenWeatherMap API
   - Teploměr na sjezdovce s aktuální teplotou
   - Počítání "ovcí" (lyžařů) pomocí YOLO AI modelu
   - Barevné označování vleků podle vytížení
   - Webkamera s HLS streamem

4. GPS A LOKALIZACE
   - Uložení aktuální GPS pozice
   - Automatické centrování mapy
   - Návrat na domovskou pozici
   - Správa jednotlivých objektů

5. ADMINISTRÁTORSKÉ NÁSTROJE
   - Přihlašovací systém s různými úrovněmi přístupu
   - Kalkulačka vlhké teploty pro zasněžování
   - Výpočet spotřeby vody v retenční nádrži
   - Správa databáze objektů
   - Statistiky a grafy

6. POKROČILÉ FUNKCE
   - Asynchronní načítání nadmořských výšek
   - Cache pro optimalizaci výkonu
   - Fallback řešení při nedostupnosti API
   - Mobilní optimalizace
   - Drag & drop pro vrstvy

TECHNOLOGIE
-----------
- Frontend: HTML5, CSS3, JavaScript (ES6+)
- Mapová knihovna: Leaflet.js
- Backend: PHP 7.4+
- Databáze: MySQL 5.7+
- AI: YOLO v8 pro detekci objektů
- API: OpenWeatherMap, OpenElevation, Mapy.cz
- Video: HLS streaming pro webkameru

KLÍČOVÉ MODULY
--------------

1. MapManager (map.js)
   - Správa mapy a vrstev
   - Drag & drop funkcionalita
   - Layer control a pořadí vrstev
   - Webkamera s HLS streamem

2. FeaturesManager (features.js)
   - Vytváření a editace objektů
   - Nadmořské výšky z API
   - Ukládání do databáze
   - GPS funkcionalita
   - Správa barev a názvů

3. WeatherManager (weather.js)
   - Real-time počasí
   - Teploměr na sjezdovce
   - Předpověď počasí
   - Automatické aktualizace

4. SheepCounter (sheep-counter.js)
   - AI detekce lyžařů
   - Barevné označování vleků
   - Real-time monitoring
   - Statistiky vytížení

5. AuthManager (auth.js)
   - Přihlašovací systém
   - Oprávnění uživatelů
   - Admin funkcionality

BEZPEČNOST
----------
- API klíče chráněny proxy servery
- CORS omezení na povolené domény
- Debug mód vypnut v produkci
- Databázové údaje v .env souboru
- Různé úrovně přístupu pro uživatele

POUŽITÍ
-------
1. Otevřete index.html v prohlížeči
2. Přihlaste se jako admin pro plné funkce
3. Vytvářejte objekty pomocí nástrojů vpravo
4. Sledujte real-time data (počasí, lyžaři)
5. Používejte GPS pro označování pozic
6. Spravujte objekty v panelu vlevo
7. Používejte kalkulačky pro zasněžování

POZNÁMKY
--------
Projekt VYGEO představuje inovativní přístup k managementu lyžařských areálů 
kombinující moderní webové technologie s geografickými informačními systémy 
a umělou inteligencí. Aplikace je navržena tak, aby byla snadno použitelná 
pro provozovatele areálů a zároveň poskytovala pokročilé funkcionality
